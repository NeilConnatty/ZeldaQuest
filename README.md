# Zelda Quest

A re-implementation of Dragon Quest 1 combat mechanics, applied to the world and creatures of The Legend of Zelda 1.

Dragon Quest formulas and stats from https://gamefaqs.gamespot.com/nes/563408-dragon-warrior/faqs/61640.

Created for the PICO-8 fantasy console: https://www.lexaloffle.com/pico-8.php.
